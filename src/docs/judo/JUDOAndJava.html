JUDO and Java

Whats the difference between JUDO and Java?

If you are new to computer programming, you are probably wondering what the differences is between JUDO and Java. Well, JUDO is a computer program, and Java is a computer programming language.

With a computer programming language, you can write computer programs. By reading the tutorials, you will learn more about how to write Java programs.

What makes JUDO a special computer program is that is lets you write new computer programs. You type the code of your program into JUDO and run then run it.

When you write Java programs in JUDO, you do not need to write full Java programs. JUDO uses a simplified version of Java so you can learn a lot faster and do interesting things quicker.
